## 使用方法

1. **首次使用**:
   ```bash
   ./aws_mfa.sh
   ```
   脚本会自动引导您完成初始设置。

2. **更新长期凭证**:
   ```bash
   ./aws_mfa.sh --update-credentials
   ```

3. **重新配置**:
   ```bash
   ./aws_mfa.sh --setup
   ```

4. **使用不同配置文件**:
   ```bash
   ./aws_mfa.sh --profile prod
   ```
